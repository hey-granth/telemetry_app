Getting Started

Follow this guide to set up the Telemetry App development environment on your local machine.

Prerequisites

Ensure you have the following tools installed:

  • Docker & Docker Compose
  • Python 3.11+ (for backend dev)
  • Flutter SDK (Stable channel)
  • Git

Backend Setup

The backend services (API + Database) are containerized for easy setup.

1. Environment Configuration

Navigate to the backend directory and create your environment file:

cd backend
cp .env.example .env
Review the .env file and update any secrets or ports if necessary.

2. Start Services

Run the following command to start the backend services:

docker-compose up -d --build

The API will be available at http://localhost:8000.

Frontend Setup

The Flutter client runs on iOS, Android, and Web.

1. Install Dependencies

cd flutter_client
flutter pub get

2. Run the App

To run on Chrome (Web):

flutter run -d chrome

To run on a connected device or emulator:

flutter run

Verification

To verify the setup is working:

  1. Open the backend API docs at http://localhost:8000/docs.
  2. Try the GET /api/v1/health endpoint. It should return { "status": "healthy" }.
  3. Launch the Flutter app and ensure it connects without network errors.
